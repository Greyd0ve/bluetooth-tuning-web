# Bluetooth Tuning Web - One Click Version

## How to start

1. Make sure Node.js LTS is installed.
2. Double-click `start.bat`.
3. The browser will open `http://localhost:5173/` automatically.
4. Keep the terminal window open while using the website.
5. To stop the website, press `Ctrl + C` in the terminal window.

## Notes

- You do not need to run `npm install` manually. `start.bat` will install dependencies automatically on first run.
- Use Chrome or Edge for Web Bluetooth.
- Use `http://localhost:5173/`, not the LAN address, for Bluetooth compatibility.

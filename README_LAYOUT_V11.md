# v11 手持布局优化说明

本版本在 v10 的基础上只调整前端布局，不改变协议、串口、BLE、本地桥的通信逻辑，适合直接继续部署到 GitHub Pages。

## 主要变化

1. 首页顶部改为仪表化布局：标题区、连接区、快速状态栏分离，适合 5 寸 / 7 寸触摸屏。
2. 新增状态卡片：连接状态、曲线点数、目标/实际速度、安全提示。
3. 导航优先级调整：联调、曲线、串口、驾驶放在前面，常用入口更靠前。
4. 连接与通用设置默认折叠，减少调参时滚动干扰；点击“展开”可修改串口、BLE、换行、节流等设置。
5. 小车联调页改为左侧 PID/摇杆、右侧大曲线，宽屏下更像一台调参仪。
6. 串口调试页改为接收区大、发送区小，更符合现场调试场景。
7. 小屏下导航栏吸顶横向滚动，急停按钮固定在右下角，便于触摸操作。

## 建议测试分辨率

- 1024x600：常见 7 寸 HDMI 触摸屏
- 800x480：常见 5 寸触摸屏
- 1280x720：便携显示器或横屏平板
- 1920x1080：普通电脑浏览器

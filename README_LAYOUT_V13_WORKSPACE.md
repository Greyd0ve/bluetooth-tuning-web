# v13 桌面联调工作台改版说明

本版本基于 v12 曲线名称编辑优化版继续修改，目标是把默认界面从多 tab 工具页调整为桌面端三栏联调工作台。

## 主要变化

1. 默认入口改为 `workspace` 工作台。
2. 顶部状态栏增强为联调状态栏，显示连接方式、设备名、连接时长、最近收包、曲线更新、PID 未发送状态和急停状态。
3. 工作台采用三栏布局：
   - 左侧：连接摘要、参数概览、PID 参数工作台、快捷动作、参数组列表。
   - 中央：紧凑曲线工具条、大曲线主区、统计条、折叠驾驶小窗。
   - 右侧：日志摘要、通道管理、实时状态。
4. PID 参数修改后会出现“未发送”视觉反馈。
5. 单项发送、分组发送、发送全部 PID 后会清除对应未发送状态。
6. 日志摘要和通道名称编辑常驻右侧栏，不再只能进单独曲线/串口页查看。
7. 原有高级功能没有删除，统一收纳到“工具”页：完整串口、曲线完整版、显示屏、按键、滑杆、记录、外观说明。
8. BLE / Web Serial / Orange Pi 本地桥通信逻辑和协议格式未改动。

## 注意

- 本版本重点优化桌面端和 Orange Pi HDMI 大屏体验。
- 小于 1280px 时会自动降级为单栏/纵向布局。
- `node_modules/` 和 `dist/` 不应提交到仓库；建议继续使用 GitHub Actions 构建。
